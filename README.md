⚽ Football Scout App

A web application that helps football scouts discover, manage, and evaluate players.
---
🚀 Features
- Player registration and profiles
- Search players by name, age, or position
- Upload player images
- Admin dashboard for managing players
- Database integration with Flask
---
🛠️ Tech Stack
- Python (Flask)
- SQLAlchemy
- HTML, CSS, Bootstrap
- MySQL
---
▶️ How to Run the Project
1. Clone the repository
2. Create virtual environment
3. Activate environment
4. Install dependencies
5. Run the app
